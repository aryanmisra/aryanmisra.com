    What it does:

    NeuraScale, the fancy name we gave our project, is a Super Resolution Generative Adversarial Network (SRGAN) with the purpose of upscaling image resolutions by a factor of 1.6 using generative deep learning. This way, a picture which initially appears blurry can be upscaled to a higher resolution.


    If you want to learn more about super resolution usings GANs, check out my article on them, here.


    All the code is available on this Github repo, as well as instructions for setting the project up.
